Features of Tungsten
We provide you the best features in Tungsten!
Overview

Tungsten is an open source, fully customizable Discord bot that is constantly growing. He comes packaged with a variety of commands and a multitude of settings that can be tailored to your server's specific needs. His codebase also serves as a base framework to easily create Discord bots of all kinds.

Fun commands

Tungsten comes loaded with tons of fun commands to play around with and increase the engagement in your server! Some commands like 8ball, catfact, dogfact, shibe, trivia, meme, yomomma, thouarttrumptweet, yesno, etc. are included in Tungsten.

Moderation

Loaded with very powerful moderation features, Tungsten is the only bot you'd ever need for your server. You can set welcome and leave channels for your server, set the starboard channel, ban/kick users, put a slowmode, warn someone and a lot other things with Tungsten.

Color role commands

Creating color roles with custom hex codes has never been easier! Create roles and assign them to users using the bot's commands!

Leveling System

Tungsten has leveling system for the chat and voice channels as well. The winner with the most points at the end of the week gets The Crown role!

Miscellaneous

You can give me your feedback using the feedback command, which would be really appreciated! In case you find any bugs, you can report that using the bugreport command and I will look into it and fix it as soon as possible.

Logging System

Tungsten has loads of logging systems, ranging from Moderator log, Member log and nickname log to Role log, Message edit log and Message delete log. You can setup the system messages for future upadtes and bug fixes of Tungsten.
